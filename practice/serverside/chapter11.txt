■事前準備
MySQLへログイン後、以下のSQLを実行して事前準備を行ってください
・データベースの作成
　CREATE DATABASE dbtest;
・データベースの変更
 USE dbtest
・テーブルの作成
　CREATE TABLE user(
	id serial primary key,
	name text,
	pass text
 );
・データ登録
 INSERT INTO user(name,pass) VALUES ('sht','pass');
