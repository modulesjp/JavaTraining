### <a name="sec1"></a>Stringクラス
Stringは基本データ型と思われがちですが、オブジェクト参照型のデータ型になります。  
Stringクラスのオブジェクトを生成するには、これまで`String 参照変数名 = "文字列";`のように定義してきましたが、以下のように記述することもできます。

```sh
char chara[] = {'I','4','U'};
String str = new String(chara);
```
